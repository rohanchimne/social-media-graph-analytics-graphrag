# Neo4j Graph Import Instructions

This document explains how to recreate the Neo4j graph used in the project.

---

# Step 1 — Start Neo4j

Start a local Neo4j instance.

Example connection:
neo4j://127.0.0.1:7687



---

# Step 2 — Prepare Data

The processed datasets generated by the notebook are used for graph creation.

Important files:

| File | Description |
|-----|-------------|
| user_tag_adoption_FINAL.csv | User-tag adoption events |
| user_similarity_topk_strict.csv | User similarity edges |

---

# Step 3 — Import Nodes

Create user nodes:

```cypher
LOAD CSV WITH HEADERS FROM 'file:///users.csv' AS row
MERGE (u:User {userId: toInteger(row.userId)});


Create tag nodes:
LOAD CSV WITH HEADERS FROM 'file:///tags.csv' AS row
MERGE (t:Tag {name: row.tag});


---

# Step 3 - Import Relationships
Create tag adoption relationships:

LOAD CSV WITH HEADERS FROM 'file:///user_tag_adoption_FINAL.csv' AS row
MATCH (u:User {userId: toInteger(row.userId)})
MERGE (t:Tag {name: row.tag})
CREATE (u)-[:ADOPTED {timestamp: row.timestamp}]->(t);


Create similarity relationships:
LOAD CSV WITH HEADERS FROM 'file:///user_similarity_topk_strict.csv' AS row
MATCH (u1:User {userId: toInteger(row.u1)})
MATCH (u2:User {userId: toInteger(row.u2)})
CREATE (u1)-[:SIMILAR_TO {
    jaccard: toFloat(row.jaccard),
    shared_tags: toInteger(row.shared_tags)
}]->(u2);


--- 

# Step 5 — Verify Graph

Count users:

MATCH (u:User)
RETURN count(u);

Count similarity edges:

MATCH ()-[r:SIMILAR_TO]-()
RETURN count(r);

Count adoption relationships:

MATCH (:User)-[r:ADOPTED]->(:Tag)
RETURN count(r);
